# Native Object Stream Class Wrappers

Node.Js ( 8.5+ ) native, dependency free, ( Object ) transfrom stream class wrappers.

## Installation
```
npm i -s class-streams
```

## Usage
```javascript
const { ObjectStream, FilterTransform } = require('class-streams');
```
